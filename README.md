# GARCH-1-1-MODEL-
GARCH(1,1) volatility model fit to 18 years of real S&amp;P 500 daily returns (2007–2026). Captures volatility clustering by forecasting variance from a baseline, recent shocks, and persistence. Built in Python with yfinance and arch, it charts a ±2σ band that widens in the 2008 and 2020 crises. Includes a live interactive dashboard. Educational only.
